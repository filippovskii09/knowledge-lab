Як читати цю схему для статті:

  • Стек (Call Stack): Це черга виконання. Коли функція викликається, вона "падає" на стек зверху. Коли завершується — видаляється.
  • Lexical Environment: Це "коробка" всередині стеку. Вона містить Environment Record (список змінних) та посилання на Outer Env (на схемі позначено стрілкою вгору).
  • Environment Record:
    • Примітиви (числа, рядки) записуються прямо в таблицю.
    • Об'єкти записують лише свою адресу (Pointer).
  • Купа (Heap): Це вільна територія. Об'єкти живуть тут, бо вони можуть змінювати розмір, і JS не знає заздалегідь, скільки пам'яті їм знадобиться.
